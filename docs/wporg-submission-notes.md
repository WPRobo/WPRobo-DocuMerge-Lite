WPRobo DocuMerge Lite is a free document automation plugin that generates PDF/DOCX documents from WordPress form submissions.

HOW IT WORKS:
Users upload a Word/DOCX template with merge tags like {client_name}, build a form using the drag-and-drop builder, and embed it via shortcode or Gutenberg block. On every submission, the plugin replaces merge tags with submitted data and delivers a completed PDF or DOCX document for download.

KEY FEATURES (FREE):
- DOCX template upload with automatic merge tag detection
- Drag-and-drop form builder with 9 field types (Text, Textarea, Email, Phone, Number, Date, Dropdown, Radio, Checkbox)
- PDF and DOCX document generation (via PHPWord + mPDF)
- Browser download delivery
- Submissions archive with document management
- Import/Export (JSON)
- Gutenberg block + shortcode for embedding
- 55 developer hooks for extensibility
- Full translation support (661 strings, .pot included)

PRO VERSION:
A Pro version (sold separately on our website) adds signature fields, Stripe payments, multi-step forms, conditional logic, email delivery, reCAPTCHA, form plugin integrations (WPForms, CF7, Gravity Forms, Fluent Forms), Elementor widget, analytics dashboard, and more. The Lite version shows Pro features as locked with upgrade prompts — no Pro code is included in the Lite plugin.

LIBRARIES INCLUDED (all GPL-compatible):
- PHPWord (LGPL-2.1) — DOCX template processing
- mPDF (GPL-2.0) — PDF generation
- Select2 (MIT) — enhanced dropdowns
- Flatpickr (MIT) — date picker
- intl-tel-input (MIT) — phone field formatting

TESTED WITH:
- WordPress 6.8
- PHP 7.4, 8.0, 8.1, 8.2, 8.3
- Plugin Check plugin (all issues resolved)
